Mobile Web

Mobile Web Group Project 2015

Welcome to our Mobile Web Group project! This website can be viewed via desktop, but was made with the mobile user in mind as it uses the JQuery Mobile framework extensively. This website displays specific information about bike stations pulled from an XML file. With this information we have derived a station list, detailed information on each station (name, city, etc), a station map using the Google Maps API, the total docks per station, the available docks at the stations and number of bikes per station. All information used is dynamically extracted from the XML file.
